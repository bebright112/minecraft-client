커스팀 기초 클라이언트

커스텀 기초 클라이언트는 뭔가요?

커스텀 기초 클라이언트는 많은 대형 서버에서 쓰이는 전용 클라이언트의 베이스 버전입니다.
간단한 파이썬 코드로 제작했으나 깔끔한 디자인과 편리한 사용성이 장점입니다.
누구나 공짜로 사용하고, 수정하실 수 있습니다.

요구사항

  • 파이썬: 파이썬을 필요로 합니다. 최신 버전을 권장합니다.
    • 파이썬 설치 바로가기
  • 자바: 자바 필요로 합니다. 최신 버전을 권장합니다.
    • 자바 설치 바로가기
  • Visual Studio Code: 편리한 수정을 위해 Visual Studio Code 설치를 권장합니다. 최신 버전을 권장합니다.
    • 설치 바로가기

사용법

  • 기초: 서버 정보는 파이썬 파일에서 수정 가능합니다. 파이썬 다운로드가 필요합니다.
  • 서버 연결: 서버의 IP 주소와 포트를 넣으세요. 포트는 0으로 입력시 생략 가능합니다. 약간의 수정이 필요할 수 있는 코드입니다.
  • 튜토리얼: 사이트를 필요로합니다. 첨부된 파일을 사용하시면 됩니다.
  • 디스코드: 서버의 디스코드로 즉시 연결할 수 있는 버튼입니다. 서버의 디스코드 링크를 넣으시면 됩니다.
